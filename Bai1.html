Hướng Dẫn Cài Đặt Python & VS Code

Hướng dẫn chi tiết từng bước để thiết lập môi trường lập trình Python hoàn chỉnh

🐍

Cài đặt Python

Ngôn ngữ lập trình phổ biến và dễ học

1

Tải Python

Truy cập trang web chính thức của Python để tải phiên bản mới nhất.

Tải Python
2

Chạy trình cài đặt

Mở file đã tải và làm theo các bước sau:

✓ Tích chọn Add Python to PATH
✓ Nhấn Install Now
✓ Đợi quá trình cài đặt hoàn tất
3

Kiểm tra cài đặt

Mở Command Prompt (CMD) và nhập lệnh sau:

❯ Command Prompt
python --version
# Kết quả: Python 3.x.x
💻

Cài đặt Visual Studio Code

Trình soạn thảo code mạnh mẽ và miễn phí

1

Tải VS Code

Truy cập trang web chính thức của Visual Studio Code.

Tải VS Code
2

Cài đặt VS Code

Chạy file cài đặt và chọn các tùy chọn sau:

✓ Tích chọn Add to PATH
✓ Tích chọn Register Code as an editor
✓ Tích chọn Add "Open with Code" action
3

Khởi động VS Code

Sau khi cài đặt, mở VS Code từ:

🔍 Tìm kiếm "VS Code"
📁 Chuột phải → Open with Code
⌨️ Gõ "code" trong CMD
⚙️

Cấu hình môi trường

Thiết lập VS Code để lập trình Python

1

Cài đặt Python Extension

Trong VS Code, cài đặt extension Python:

1. Nhấn Ctrl + Shift + X để mở Extensions
2. Tìm kiếm "Python"
3. Chọn extension của Microsoft và nhấn Install
2

Tạo file Python đầu tiên

Tạo một file mới với đuôi .py:

📄 hello.py
print("Xin chào, Python!")
3

Chạy code Python

Có nhiều cách để chạy code Python trong VS Code:

▶️ Run Button

Nhấn nút Run ở góc trên bên phải

⌨️ Phím tắt

Nhấn F5 hoặc Ctrl+F5
